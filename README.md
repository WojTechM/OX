# About OX
Tic-Tac-Toe game

## Prerequisities:
Maven 3.6 , Java 1.8

## Running: 
To run the app you can use one of "compileAndRun" scripts

Important note: you might need to use "chmod +x" on ".sh" files in order to use them.


#### With Tests:
./compileAndRunWithTests.sh
#### Without Tests:
./compileAndRun.sh
